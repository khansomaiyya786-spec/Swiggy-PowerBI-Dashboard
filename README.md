# Swiggy Business Performance Dashboard

## Project Overview

This Power BI dashboard analyzes Swiggy business performance using interactive visualizations and KPIs.

## Features

- Total Orders
- Gross Order Value (GOV)
- Monthly Active Users
- Average Delivery Time
- Platform Rating
- Monthly Revenue Trends
- Orders by City
- Category Analysis
- Top Restaurant Partners

## Tools Used

- Power BI
- Power Query
- DAX
- Excel
